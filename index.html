  4. Get a temporary token: gcloud auth print-access-token
  5. Paste the token into the API Key field.
  6. Use API URL built by the Build Vertex AI URL button and model google/gemini-2.5-flash.

Pipeline rules
  1. Entity cleaning removes Markdown code fences and separates invalid JSON into an error CSV.
  2. TEI export requires checked source text and marks entities found in that text.
  3. Gazetteer export leaves normalization, identifiers, and coordinates blank for authoritative lookup.
  4. Network export requires reviewed relationship JSON with evidence; co-occurrence alone never creates an edge.
